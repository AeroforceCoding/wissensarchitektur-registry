# Registry-Workflow-Prompts

Dieser Bereich enthält Prompts, die innerhalb der Wissensarchitektur-Registry den kanonischen Verarbeitungsprozess steuern.

Die Ordner folgen der operativen Reihenfolge:

1. `01_eingang`
2. `02_zuordnung`
3. `03_pruefung`
4. `04_entscheidung`
5. `05_aktualisierung`
6. `06_thebrain`

Diese Prompts sind primär für die Arbeit im Repository gedacht. Extern kopierbare Chat- oder Repo-Anweisungen liegen getrennt unter `prompts/externe_anwendung/`.
