# Externe Anwendungsprompts

Dieser Bereich enthält Prompts und Anweisungen, die außerhalb des unmittelbaren Registry-Workflows verwendet werden können.

## `chat_anweisungen`

Anweisungen für neue oder bereits laufende Arbeitschats. Sie helfen dabei, Erkenntnisse so zu führen, dass sie später per Chat-Extraktion in die Registry überführt werden können.

## `repo_erweiterungen`

Prompts für die strukturierte Erweiterung anderer Repositories oder angrenzender Arbeitsräume.

Diese Prompts bleiben trotzdem Teil der kanonischen Prompt-Bibliothek dieses Repositories. Änderungen erfolgen daher zuerst hier und werden danach bei Bedarf in andere Systeme übernommen.
